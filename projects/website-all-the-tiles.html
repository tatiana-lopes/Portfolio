All The Tiles

Live website

This is a Full-stack group project which has two websites in one. The first All The Tiles is a website with a gallery of generated tiles by users made from the second website Tile Creator which allows users to create their own generated tiles designs. All The Tiles was made in Vercel with Cosmic.js and React framework. The Tile Creator was made with P5.js. My role was to design the websites with Figma and then implement the Tile Creator website and contribute to the All The Tiles website.
